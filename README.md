# Medium
### Vue.js frontend application
Учебный проект клон сайта [Medium realworld](https://demo.realworld.io/)
Используется готовый API https://conduit.productionready.io/api

####В проекте используется:

-Vue cli <br>
-Vuex <br>
-Vue Router <br>
-Axios <br>
-Bootstrap <br>
